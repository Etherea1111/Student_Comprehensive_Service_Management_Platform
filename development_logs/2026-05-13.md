## 当前运行方式

小程序运行：

```bash
使用微信开发者工具导入仓库根目录
```

后端本地启动建议：

```bash
cd backend
npm install
cp .env.example .env
npm run db:init
npm run dev
```

小程序连接后端：

```js
// miniprogram/config/env.js
module.exports = {
  apiBaseUrl: 'https://你的后端域名/api',
  tokenStorageKey: 'student_service_token'
}
```
